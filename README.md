🥋 神之一手：技能五子棋 (God's Hand: Skill Gomoku)“方寸棋盘，指点江山。一子落下，风云变色。
”神之一手 是一款融合了 RPG 角色扮演 与 卡牌技能 的创新五子棋游戏。在禅意十足的水墨画风中，扮演热血少年或绝世大师，利用强力的技能卡牌扭转战局，体验不一样的弈棋之道。✨ 核心特色技能卡牌系统：告别传统枯燥的你来我往，抽取 N、SR、SSR 不同稀有度的技能卡，打出意想不到的连招。

角色扮演体验：
热血少年：初生牛犊不怕虎，誓要挑战武林巅峰。
绝世大师：深藏不露的扫地僧，举手投足皆是宗师气派。

禅意水墨视效：古朴的宣纸背景、动态水波纹落子特效、震撼的水墨大字结算。

智能 AI 对手：无论你扮演谁，AI 都将化身你的对手，不仅懂得常规定式，更懂得如何应对（或使用）技能。

单机离线畅玩：纯原生 HTML5/JS 开发，单文件运行，下载即玩，无需联网。

🎮 玩法说明
1. 游戏开始打开游戏后，在主界面选择你想扮演的角色：选择少年：你将扮演少年，对抗由 AI 控制的大师。选择大师：你将扮演大师，教训由 AI 控制的少年。注：无论选择哪个角色，玩家永远执黑先手（Player 1）。
2. 基础操作落子：点击棋盘交叉点即可下棋。使用技能：点击左侧你的手牌，高亮选中后，根据技能描述进行操作（如点击棋盘特定位置）。再次点击手牌：取消选中状态，恢复普通落子模式。
3. 胜利条件：五子连珠：横、竖、斜任意方向连成 5 子。技能获胜：触发 SSR 技能特殊胜利条件。
   🃏 技能卡牌图鉴游戏包含三种稀有度的技能卡牌，策略使用是制胜关键：
   🟢 N 卡：飞沙走石 (Flying Sand)效果：移除对手的一颗棋子，并且该位置在下一回合变为“禁手”。禁手规则 (v6.1 新增)：被移除棋子的位置会留下红色标记，对手在接下来的一回合内无法填补该位置。这是切断对手棋路、破坏防守的利器。消耗：一次性消耗。
   🔵 SR 卡：双龙戏珠 (Double Dragon)效果：本回合可以连续落下两子。限制：两子不能重叠。战术：瞬间制造“双活三”或“四三杀”的绝佳手段。
   🔴 SSR 卡：力拔山兮 (Conqueror's Power)效果：终极必杀技。如果棋盘上己方存在任意**“活三”（两头无阻挡的三连子），直接判定游戏胜利**。战术：无需苦凑五子，只要造出活三，一击必杀。
